Sobre Mim

Oi, sou Washington! Estou começando minha jornada na programação e, para ser sincero, estou me divertindo muito. Entrei no Oracle One para aprender o básico de HTML, CSS e JavaScript, e, apesar de ainda não me sentir 100% confiante, já percebo o quanto evoluí desde que comecei.

O mundo da programação é desafiador, sem dúvida, mas é esse desafio que me atrai. Resolver problemas, quebrar a cabeça com o código e, no final, ver o que criei funcionando – não tem sensação melhor.

Atualmente, ainda não me aprofundei em frameworks como o React, mas já estou ansioso para dar esse próximo passo. E o melhor: sei que cada código escrito é uma conquista, mesmo que pequena.

Esse site é um exemplo do que já consegui fazer até agora.
